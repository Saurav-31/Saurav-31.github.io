# Saurav Gupta - Academic Portfolio

Personal academic website built with [Hugo Blox Builder](https://hugoblox.com) and the Academic CV theme.

## About

This website showcases my research, projects, and publications in Computer Vision, Deep Learning, and Automotive AI.

## Local Development

To run this site locally:

```bash
# Install Hugo
# See: https://docs.hugoblox.com/start/cli/

# Run the development server
hugo server
```

The site will be available at `http://localhost:1313`

## Deployment

This site is configured for deployment on GitHub Pages. The site will automatically build and deploy when changes are pushed to the main branch.
